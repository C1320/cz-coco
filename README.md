# cz-coco

vue3.0 UI

基于 Vue3.x + element-plus + Scss 封装的基础组件库

添加了 tailwindcss 支持
## install

```bash
npm install
# OR
yarn install
```

## dev

```bash
npm run docs:dev
# OR
yarn docs:dev
```

## build docs

```bash
npm run docs:build
# OR
yarn docs:build
```

## build components

```bash
npm run build
# OR
yarn build
```

[组件库交互文档](https://leitingting08.github.io/cz-coco/)
