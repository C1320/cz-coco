# @cz-coco/layout

## Install

```bash
npm i @cz-coco/layout
# or
yarn add @cz-coco/layout
```

## Usage

```vue
<template>
    <sum-layout />
</template>

<script lang="ts">
import { defineComponent, toRefs } from 'vue'
import SumLayout from '@cz-coco/layout'

export default defineComponent({
    components: { SumLayout },
    setup() {}
})
</script>
```
